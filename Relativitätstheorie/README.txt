Urheber

Dies ist eine von mir erstellte Simulation zur Veranschaulichung von Längenkontraktion und Zeitdilatation aufgrund der Beschleunigung in der Relativitätstheorie.
Das UFO-Bild ist von Pixabay: https://pixabay.com/de/vectors/ufo-alien-schiff-raumschiff-4778062/
Die Plotting-Bibliothek heißt Plotly: https://plotly.com/
Diese kann ich Ihnen wärmestens ans Herz legen, da sie alle möglichen 2D- und 3D-Plots und -Animationen ermöglicht.
Den Code zum Downloaden der SVG ist von einem Codepen-Post: https://codepen.io/Alexander9111/pen/VwLaaPe?editors=1010
Sie können Geschwindigkeit und Ruehmasse des Objektes einstellen.
Dann werden in Echtzeit die Zeitdilatation und die kinetische Energie angezeigt.
Die Geschwindigkeit kann per Slider.
Die Masse kann auch mit dem entsprechen Slider verstellt werden.
Mit "c" oder einem Klick auf das Raumschiff kann die Simulation aus dessen Sicht betrachtet werden.
Dabei ist zu bedenken, dass bei der Darstellung der Sterne keine relativistischen Effekte miteinbezogen werden.
Mit "d" können Sie das Diagramm als SVG herunterladen.
